# 💖 Hatimo Backend

**하티모(Hatimo)** 는 `heart` + `emotion`을 합친 이름으로,  
사용자의 감정을 기록하고 AI가 따뜻하게 위로해주는 감정 기록 서비스입니다.

이 저장소는 하티모의 **백엔드(Spring Boot)** 서버를 담당합니다.

---

## 🚀 프로젝트 정보

| 항목             | 내용                            |
|------------------|---------------------------------|
| 개발 언어        | Java 17                         |
| 프레임워크       | Spring Boot 3.4.4              |
| 빌드 도구        | Gradle                          |
| 데이터베이스     | PostgreSQL                      |
| API 연동         | OpenAI GPT (gpt-3.5-turbo 예정) |
